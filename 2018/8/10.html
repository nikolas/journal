The new MediaWiki 1.31.0 version (live on Wikipedia) makes a change to how the "Edit source" button behaves. Instead of opening up a simple <textarea&rt;, the source markup is loaded in the page with JavaScript, just like how the new Visual Editor works.

I can see how the new system is more orthogonal and might simplify some code: Now both editors instantiate themselves in the same way. The new source editor is slower, though.
